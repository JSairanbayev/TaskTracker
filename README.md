# TaskTracker

Console task tracking application written in C++ with status management and persistent file storage.

## Features

- Add tasks
- View all tasks with status
- Change task status (New / In Progress / Done)
- Filter tasks by status
- Save and load tasks from file

## Technologies

- C++
- OOP (classes, encapsulation, setters)
- Enum types for status management
- STL (vector, string)
- File I/O (fstream)
- Input validation

## How to run

1. Clone the repository
2. Open `TaskTracker.cpp` in Visual Studio or any C++ compiler
3. Build and run
4. Tasks are saved to `tasks.txt` in the program directory
